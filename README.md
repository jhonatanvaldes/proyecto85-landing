# Proyecto 85 Landing

Landing page premium para Proyecto 85, evaluación médica de longevidad preventiva del Dr. Jhonatan Valdés.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Desplegar

El proyecto está listo para Vercel como una app Next.js App Router con TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, lucide-react y next/image.
